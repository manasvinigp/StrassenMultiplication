What is Strassen's Matrix Multiplication?

Divide-and-Conquer Strategy:

  • Divide: Break down the input matrices into smaller submatrices.
  • Conquer: Multiply these submatrices recursively until reaching a base case.
  • Combine: Combine the results of the smaller multiplications to obtain the final result.

Strassen's Algorithm Steps:

  1. Divide Step:

    Given two matrices A and B, divide each matrix into four equal-sized submatrices:
    A = [[A11, A12], [A21, A22]]
    B = [[B11, B12], [B21, B22]]

  2. Conquer Step:

    Recursively compute seven matrix products:
    • F1 = A11(B12-B22)
    • F2 = (A11+A12)B22
    • F3 = (A21+A22)B11
    • F4 = A22(B21-B11)
    • F5 = (A11+A22)(B11+B22)
    • F6 = (A12-A22)(B21+B22)
    • F7 = (A11-A21)(B11+B12)

  3. Combine Step:

    Compute the resulting submatrices of the product matrix C using the following equations:
    • C11 = F5 + F4 - F2 + F6
    • C12 = F1 + F2
    • C21 = F3 + F4
    • C22 = F5 + F1 - F3 - F7
